## PositionGroup

Supported groups:

```text
QB, RB, WR, TE, OT, IOL/C, EDGE, IDL, LB, CB, S, ST
```

These are fixed because the app is a current Raiders roster tool, not a generic roster builder.

## SourceRef

```ts
type SourceRef = {
  label: string;
  url: string;
  retrievedAt: string;
  note?: string;
};
```

Use `SourceRef` on every seed row that depends on public or local source material.

## RosterPlayer

```ts
type RosterPlayer = {
  name: string;
  position: string;
  positionGroup: PositionGroup;
  positionFlex: string[];
  age: number | null;
  experience: string;
  college: string;
  sourceRefs: SourceRef[];
};
```

Roster data describes official roster identity and flexible role capability. It does not declare official depth order.

## SchemeRole

```ts
type SchemeRole = {
  id: string;
  label: string;
  positionGroup: PositionGroup;
  schemeTraits: string[];
  dependencyScore: number;
  starterLeverage: number;
  replacementScarcity: number;
  failureModes: string[];
  sourceRefs: SourceRef[];
};
```

Scheme roles describe what the Raiders need each position group to preserve.

## DepthAssessment

```ts
type DepthAssessment = {
  positionGroup: PositionGroup;
  starter: string;
  backups: string[];
  crossTrainedPlayers: string[];
  crossTrainingCoverage: number;
  depthDropoff: number;
  confidence: number;
  whatBreaks: string;
  sourceRefs: SourceRef[];
};
```

Depth assessments are inferred until an official depth chart exists.

## Prospect

```ts
type Prospect = {
  name: string;
  school: string;
  position: string;
  positionGroup: PositionGroup;
  projectedRange: { min: number; max: number };
  consensusRank: number | null;
  nflIqOverall: number;
  athleticism: number;
  production: number;
  readiness: number;
  roleFits: Partial<Record<PositionGroup, number>>;
  traits: string[];
  sourceRefs: SourceRef[];
  status: "available" | "drafted";
};
```

Prospects can fit more than one position group through `roleFits`, but the main recommendation flow uses the prospect's primary `positionGroup`.
