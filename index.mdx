Raiders Depth Fragility is a draft-room decision support app for one question:

> Who should the Raiders consider next because he most reduces current Super Bowl-limiting fragility?

The app does not try to replace scouting grades or a consensus board. It adds a roster-specific layer: if a starter goes down, which position groups lose the ability to execute the current Raiders scheme, and which available prospects most reduce that failure risk?

<Warning>
This project is intentionally current-state only. There is no season selector, historical mode, or generic big-board workflow.
</Warning>

## What the tool ranks

The app produces two linked rankings.

  <Card title="Prospect fit" icon="target" href="/model/prospect-fit">

## Current facts encoded in the build

The seed data is timestamped at `2026-04-25T02:32:16-05:00` and uses public source references embedded in every row.

- Day 2 of the 2026 NFL Draft was April 24, 2026.
- Fernando Mendoza is treated as already drafted by Las Vegas at No. 1 overall.
- Treydan Stukes, Keyron Crawford, and Trey Zuhn III are treated as already drafted by Las Vegas.
- The active decision window starts at the next live Raiders pick, currently No. 102.
- The official Raiders depth chart is unavailable until Fall 2026, so depth order is inferred and shown with confidence warnings.
- NFL IQ is treated as the primary conceptual source, with a manual CSV path for values that are visible to a user but not available through a stable public API.
